SwipePanel demo project

This is a simple demo that shows the swipeable control in action. It shows how controls can be embedded in a swipeable panel as well as how to add and select pages at runtime.

Panel 1
Panel 2
Panel 3
The control has 2 properties:
.pageIndex - The currently selected page (0 - n)
.pageCount - The total number of pages (n)

The control has 1 public method:
.addPage(x) - Adds a page to the control
              Element 'x' must be a <div>
